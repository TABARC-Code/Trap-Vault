# Idiots Guide to Trap Vault

This is the simplest possible explanation of what this system is and how to use it without overthinking.

## What this actually is

Trap Vault is not a notes app.

It is a thinking system where:
- you generate ideas
- Claude structures them
- the vault organises them into something reusable

## The loop

1. Dump raw input (notes, links, thoughts)
2. Ask Claude to process it
3. Claude outputs structured markdown
4. You store it in the correct folder
5. Over time patterns emerge

## The rule

Do not organise manually if Claude can do it better.

## Minimal workflow

- Capture -> /00_inbox/
- Process -> Claude
- Store -> /01_concepts/, /02_datasets/, /03_synthesis/

That is it. Everything else is optimisation.
