# Notes (Working Log)

This is less documentation, more thinking out loud.

If you're looking for polished explanations, wrong place.
If you're trying to track how the idea mutates over time, this is it.

---

## Entry 01 — The shift

Most AI work is obsessed with answers.

Feels like the wrong layer.

The more interesting question is:

> where does a thought go before it becomes an answer?

That’s what this system is poking at.

---

## Entry 02 — Dewey wasn’t about books

Dewey Decimal is usually treated like a storage system.

It isn’t.

It’s a compression scheme for human knowledge.

Each number is doing two things at once:

- categorising
- implying relationships

That second bit is the bit most systems ignore.

---

## Entry 03 — Models don’t organise themselves

LLMs are brilliant at:

- pattern completion
- language synthesis

They’re terrible at:

- stable internal structure
- consistent categorisation

So instead of forcing the model to be organised, the plan is to bolt structure around it.

---

## Entry 04 — Classification might be intelligence

Working hypothesis:

> intelligence = classification × recursion × memory

Not proven. Not even fully defined. But it keeps showing up.

---

## Entry 05 — Problem emerging

If everything can be classified, then classification itself needs classifying.

Which is where this starts getting recursive in a slightly concerning way.

---

## Entry 06 — Direction forming

This is starting to look like a routing system, not just a knowledge system.

Ideas don’t sit still. They move.

---

## Entry 07 — Realisation

This isn’t a chatbot.

It’s closer to a cognitive filing system that happens to speak.

---

## Entry 08 — Still unclear

Big unknowns:

- how rigid the taxonomy should be
- whether nodes should mutate over time
- how to handle ambiguity without collapsing structure

Leaving these open for now.
