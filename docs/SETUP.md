# Setup

## Requirements

Minimum:
- Git
- Obsidian
- Claude Code or another file-writing Claude environment

Recommended:
- Python 3
- a terminal

## Obsidian

Open the repo folder directly as a vault.

Recommended settings:
- Attachment folder path -> `raw/assets/`
- enable backlinks
- use graph view

Optional plugins:
- Dataview
- Marp

## Claude Code

Open the same folder as a project.

## Optional local MCP scaffold

This repo includes:

```text
mcp_server/
```

You do not need it.

Use it only if you want a structured local tool layer later.
